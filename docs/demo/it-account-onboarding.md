# 员工入职 IT 账号开通流程

## 适用范围

本文适用于正式员工、实习生、外包人员和临时项目成员在入职当天开通基础办公账号的场景。账号范围包括统一身份账号、企业邮箱、飞书、VPN、财务报销系统和部分业务系统。

## 标准流程

1. HR 在员工入职前 3 个工作日完成入职信息确认。
2. 系统根据员工工号自动创建统一身份账号。
3. 员工入职当天 09:00 前收到激活短信或个人邮箱通知。
4. 员工首次登录后需要修改初始密码，并完成 MFA 绑定。
5. 基础权限会在账号激活后 30 分钟内自动同步。

## 默认开通系统

| 系统 | 默认开通 | 说明 |
| ---- | -------- | ---- |
| 飞书 | 是 | 用于 IM、日历、文档协作 |
| 企业邮箱 | 是 | 邮箱地址格式为 name@company.example |
| VPN | 否 | 需要直属主管审批 |
| 财务报销系统 | 是 | 转正员工默认开通 |
| 代码仓库 | 否 | 需要研发负责人审批 |

## VPN 权限申请

VPN 权限默认不自动开通。员工需要在权限系统提交申请，填写使用原因、访问系统、预计使用周期，并选择直属主管作为审批人。审批通过后，VPN 权限通常 15 分钟内生效；若超过 1 小时仍无法登录，需要提交 IT 工单。

## MFA 绑定要求

MFA 是访问企业邮箱、VPN、代码仓库和财务系统的基础安全要求。员工首次登录统一身份账号后必须完成 MFA 绑定。未绑定 MFA 的账号只能访问基础门户，不能访问敏感系统。

## 常见问题

### 没有收到账号激活短信怎么办？

请先确认 HR 录入的手机号是否正确。如果手机号正确但仍未收到短信，可以联系 HRIS 服务台重新发送激活通知。

### 外包人员可以开通邮箱吗？

默认不开通。若业务需要，需由业务负责人发起临时邮箱权限申请，并说明使用周期。

### VPN 权限多久生效？

VPN 权限审批通过后通常 15 分钟内生效。若超过 1 小时仍无法登录，请提交 IT 工单。

## Agent 回答提示

当用户询问入职账号、邮箱、飞书、VPN、MFA 相关问题时，优先引用本文。回答时需要说明申请入口、审批人和预计生效时间。
