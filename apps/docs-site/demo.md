## 推荐演示路径

现场演示可以按“文档入库 -> 检索 -> Agent 问答 -> 引用追溯 -> 回答标注入库”走一遍。

1. 登录系统，进入模型管理，确认 Chat 模型、Embedding 模型已经启用。
2. 创建知识库，例如“企业制度知识库”，绑定 Embedding 模型。
3. 如果要演示 Rerank 精排，继续绑定 Rerank 模型。
4. 如果要演示多模态 MVP，继续绑定视觉模型和图片理解提示词。
5. 在知识库文档页上传 Markdown、CSV、DOCX 或 PDF 样例。
6. 在上传预览页查看分段草稿，必要时编辑标题、内容和启停状态。
7. 确认入库后进入分段页，查看分段、图片引用和图片理解 Chunk。
8. 打开检索测试，提问示例问题，查看命中分段、来源和分数。
9. 创建专家 Agent，绑定 Chat 模型和该知识库。
10. 进入 Chat 页面提问，查看流式回答和引用来源。
11. 回到对话日志，把有价值的 AI 回答标注入库。
12. 查看知识热度页，观察引用次数和覆盖分段统计。

## 演示材料

仓库内置企业场景样例，适合直接上传测试：

| 文件                                         | 场景                                       | 推荐问题                          |
| -------------------------------------------- | ------------------------------------------ | --------------------------------- |
| `docs/demo/it-account-onboarding.md`         | 入职账号、邮箱、飞书、VPN、MFA             | VPN 权限多久生效？                |
| `docs/demo/procurement-contract-approval.md` | 采购合同、审批材料、金额节点、风险条款     | 采购合同超过 100 万需要哪些审批？ |
| `docs/demo/security-incident-response.md`    | 钓鱼邮件、账号异常、数据泄露、安全事件上报 | 发现钓鱼邮件应该多久内上报？      |
| `docs/demo/expense-policy-table.csv`         | 报销费用类型、限额、审批人、生效时间       | 客户招待费用的单次限额是多少？    |
| `docs/demo/media/risk-rules.docx`            | 带图片的 Word 风险规则材料                 | 规则图里如何判断高风险合同？      |

PDF / DOCX / CSV 样例覆盖多格式解析、分段预览、引用追溯和图片理解 Chunk。

## 可验收结果

- 文档上传后能进入预览页，并生成可编辑分段。
- 确认入库后，文档状态进入可检索状态。
- 检索测试能看到命中分段、来源文档和分数。
- Chat 页面能流式返回回答，并展示引用来源。
- RAG 评测页能展示 Ragas / DeepEval 的离线基准结果。
